How to use
1

Find your Xbox saves at
C:\Users\YOU\AppData\Local\Packages\HoodedHorse.SuperFantasyKingdom_...\SystemAppData\wgs\...\
There will be a subfolder per profile — drop files from all of them at once.

2

The tool groups files by profile automatically and shows what it found. Assign each detected profile to a PC save slot (1, 2, or 3).

3

Click Convert & Download Files. Each save file will download individually with the correct name. Move them into
%USERPROFILE%\AppData\LocalLow\Super Fantasy Games\Super Fantasy Kingdom\
Delete any existing .bak files in that folder first.

All processing happens in your browser — no files are uploaded anywhere.
